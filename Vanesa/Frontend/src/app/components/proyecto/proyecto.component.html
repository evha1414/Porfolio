Proyecto

Argentina Programa

Desarrollar una aplicación web full stack con datos personales, estudios cursados, experiencia laboral, conocimiento de las tecnologías. Esta aplicación deberá ser de arquitectura distribuida y contener un diseño de interface (front end) que muestre la información, una base de datos que almacene los datos antes mencionados y debe contar con las APIs necesarias para proveer a través de internet la información (back end).
